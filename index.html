Completed Games

Asteroids

Monogame And C#

This Asteroids-style game was developed in MonoGame using C# for my Master’s coursework. It features a custom object-oriented 2D bounding box (OBB) collision system designed to detect collisions between asteroids and between asteroids and the player. The game also uses multithreaded processing to improve performance and maintain smooth gameplay.

Infinirun

Unity3D and C#

This hypercasual infinite runner was created in Unity with C# during a one-day passion project. It features procedural map generation and custom shaders to deliver a visually dynamic and endlessly replayable experience.

This is not a drill

Unity2D and C#

This Is Not a Drill is a 2D survival game developed in Unity with C# for the 36-hour game jam Devs That Jam Challenge #8, where it won second place. Players must drill downward to escape a nuclear fallout, navigating a procedurally generated world built using Perlin noise.

Res8

Unity2D and C#

Res8 is a game jam project created for the BYOG (Build Your Own Game) Jam. Built in Unity with C#, it features a unique time-reversal mechanic that allows players to manipulate the timeline of specific objects, creating strategic and mind-bending gameplay scenarios.

Technical Projects

Boids ECS

Unity ECS and Jobs System

A high-performance boid simulation built in Unity using the Entity Component System (ECS) and C# Job System, capable of simulating over 30,000 autonomous agents in real time. This project leverages data-oriented design and multithreading to achieve massive scale and smooth performance, demonstrating efficient use of parallel processing and memory layout optimization.

Domain Dash

OpenGL and C++

A top-down 3D game built in C++ with OpenGL featuring physics-based motion, real-time collisions, and a custom camera system with smooth, gimbal-lock-free rotation. Models were created using Blender and procedural scripting, showcasing efficient graphics and camera design.

Rollercoaster

OpenGL and C++

A 3D simulation featuring a procedurally generated rollercoaster track using Catmull-Rom splines and motion powered by Verlet integration. Built in C++ with OpenGL, it also includes dynamic, procedurally generated sea waves and a mix of scripted and FBX-imported models to create an immersive environment.

Falling Sand

Monogame and C#

A lightweight 2D pixel-based sand simulator prototype built in MonoGame using C#. Inspired by Conway’s Game of Life, it uses cellular automata logic to simulate falling particles. The current version supports interactive user input and showcases emergent behavior through simple physics-inspired rules.
